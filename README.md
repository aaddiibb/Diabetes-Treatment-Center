# Diabetes Treatment Center

## 1. PROJECT OVERVIEW

### 1.1 Introduction
The Diabetes Treatment Center is a comprehensive JavaFX-based desktop application designed to facilitate diabetes management through predictive analytics, appointment scheduling, and health monitoring. The system serves two primary user roles: Patients and Doctors, providing role-based functionalities to enhance diabetes care delivery.

### 1.2 Objectives
- **Diabetes Risk Prediction:** Implement machine learning-based risk assessment using logistic regression
- **Appointment Management:** Enable seamless booking and management of doctor-patient appointments
- **Health History Tracking:** Maintain comprehensive health metrics and prediction history
- **PDF Report Generation:** Generate exportable health reports for medical records
- **Visual Analytics:** Provide trend charts for glucose, BMI, and risk percentages
